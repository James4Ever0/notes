# Why you want to have a mac

Mac is not necessary good.

Warp supports Linux right now, and many other "Mac exclusive" apps as well.

Tmuxai is Linux friendly.

You can run Mac in virtual machine.

You can run Mac with the trashcan model.

Mac can be running on non-apple hardware.

You can first recall a tool from Mac then seek for replacement.

THINK how you are going to improve efficiency across multiple interfaces and tasks. If you want filename searching, task switching, document reference, buffer sharing, file transfer or else? You can learn and collect good tools along the way.
