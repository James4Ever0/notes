# 焊接几大原则

尽量用高温焊锡 大功率烙铁

太细小的焊点 就不要焊接了 最好用加热台加锡膏加高温胶带 镊子进行操作 或者观察好线序之后买成品端子线

usb端子线最好万用表先测一下原来的线序 不能只看颜色

有线端子的尽量用原来的端子 不要尝试在端子上面焊接导线

尽量不要一根一根空中接线 容易搞错 或者电阻过大

如果是大功率电源线 尽量不要空中接线 在线的根部进行操作最好

焊点绝缘用高温热缩管（提前包裹） 电子硅橡胶

发热的电路外壳用耐高温阻燃热熔胶
