# 情感陪伴ai

根据你的需求，在GitHub上找到了几个高度相关的开源项目，其中 CharacterChat 能最好地满足你的所有要求。

👑 首选推荐：CharacterChat（最符合所有要求）

CharacterChat 是一个学术级别的开源项目，它几乎完美匹配你的所有需求。

· ✅ 多种MBTI选择：核心就是基于MBTI人格分解，构建了庞大的“MBTI-1024 Bank”虚拟角色库。
· ✅ 接入主流LLM与向量数据库：系统由“角色和记忆驱动的对话模型”构成，“动态记忆”的设计暗示了与向量数据库的紧密集成。
· ✅ 找回历史对话记录：其“动态记忆”和“记忆驱动”的设计，正是为了实现长期记忆和上下文连贯的对话。
· 项目地址：https://github.com/morecry/CharacterChat

---

🚀 其他高度相关的开源项目

1. Airi（人气之选）：一个在GitHub上很火爆的虚拟伴侣项目。
   · 特点：支持自定义MBTI、兴趣等；基于向量数据库实现长期记忆。
   · 项目地址：https://github.com/moeru-ai/airi
2. OpenPDB（人格即服务）：一个为AI赋予人格的工具包。
   · 特点：提供超过12,000种人格档案，包含MBTI；明确支持接入本地LLM（如Ollama）。
   · 项目地址：https://github.com/gitsual/openpdb
3. SoulForge（数字人格框架）：一个用于构建持久、有情感意识数字人格的框架。
   · 特点：支持MBTI类型等心理原型；注重“持久化”和“记忆”。
   · 项目地址：在GitHub或NPM上搜索 soulforge-framework。
4. Aibou（终端宠物）：一个基于MBTI的终端宠物伴侣。
   · 特点：包含完整的16种MBTI人格。
   · 项目地址：在GitHub或NPM上搜索 aibou。

💎 总结与建议

综合来看，如果你追求功能最全、最贴近学术前沿，CharacterChat 是首选。如果看重社区活跃度和易用性，Airi 会是很棒的选择。而 OpenPDB 则非常适合需要灵活定制多种人格的应用场景。

建议直接访问这些项目的GitHub页面，通过查看 README 文档和 Star 数来进一步评估。
